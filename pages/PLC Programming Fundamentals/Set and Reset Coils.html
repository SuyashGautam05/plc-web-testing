Set and Reset Coils

Set and Reset coils are special types of output instructions used in Programmable Logic Controllers (PLCs) to control outputs or internal memory bits in a latched manner. Unlike normal output coils that depend continuously on the logic condition, set and reset coils allow an output to remain ON or OFF even after the controlling condition is removed. This feature is extremely useful in industrial automation where memory of a previous event is required.

Set and reset coils are widely used in start-stop control circuits, sequence control, safety logic, and machine operation states.

Set Coil (SET Instruction)

The Set coil is used to turn ON (energize) an output or internal memory bit and keep it ON even after the input condition becomes false.

Working of Set Coil

When the logic condition connected to the set coil becomes true:

  • The set coil energizes the specified output or memory bit.
  • The output remains ON even if the input condition turns OFF.
  • The output stays ON until a reset instruction is executed.

The set coil is mainly used where an output must remain active after a momentary input, such as a start push button.

Key Characteristics of Set Coil

  • Provides memory (latched operation).
  • Output does not depend on continuous input.
  • Used for maintaining machine states.
  • Requires a reset instruction to turn OFF.

Reset Coil (RESET Instruction)

The Reset coil is used to turn OFF (de-energize) an output or internal memory bit that was previously set using a set coil.

Working of Reset Coil

When the logic condition connected to the reset coil becomes true:

  • The reset coil clears the specified output or memory bit.
  • The output turns OFF immediately.
  • The output remains OFF until it is set again.

Reset coils are commonly used with stop buttons, fault signals, or emergency conditions.

Key Characteristics of Reset Coil

  • Clears a previously set output.
  • Overrides the set condition.
  • Ensures safe shutdown of machines.
  • Essential for stop and fault logic.

Set-Reset Operation in PLC Logic

Set and reset coils always work together. The same output or memory address is used for both set and reset instructions.

Typical operation:

  • SET coil → Turns output ON.
  • RESET coil → Turns output OFF.

Example:

  • Start push button → SET motor output
  • Stop push button → RESET motor output

This logic ensures controlled and predictable operation of machines.
