RSLogix Micro

RSLogix Micro is a PLC programming software developed by Rockwell Automation for programming Allen-Bradley MicroLogix PLCs. It provides a reliable environment for creating, editing, and monitoring PLC programs using Ladder Diagram programming. RSLogix Micro is commonly used in small to medium automation applications and is well known for its stability and industrial acceptance.

RSLogix Micro supports MicroLogix PLC families such as MicroLogix 1000, 1100, 1200, and 1400. The software allows users to develop control logic, configure communication, and perform online monitoring and troubleshooting. Its ladder-based approach makes it easy for technicians and engineers familiar with relay logic.

Features of RSLogix Micro

  • Provides Ladder Diagram-based PLC programming environment.
  • Supports Allen-Bradley MicroLogix PLC series.
  • Allows offline program development and online program editing.
  • Provides real-time monitoring of inputs, outputs, and internal bits.
  • Supports timers, counters, comparison, math, and data handling instructions.
  • Includes diagnostic tools for fault detection and troubleshooting.

PLC Programming Using RSLogix Micro

PLC programming using RSLogix Micro is carried out by creating a project for the selected MicroLogix PLC model. Control logic is written using Ladder Diagram instructions such as contacts, coils, timers, counters, and math operations. I/O addressing is configured according to the PLC hardware.

The software supports online monitoring, where the programmer can view active ladder rungs and real-time status of PLC variables. This feature helps in debugging programs during commissioning and maintenance. Changes can be tested and verified efficiently to ensure proper system operation.

Applications of RSLogix Micro

  • Used in small and medium industrial automation systems.
  • Applied in motor control and machine automation.
  • Used in conveyor systems, packaging machines, and assembly lines.
  • Suitable for process monitoring and basic control applications.
  • Widely used in industrial plants and educational PLC laboratories.
