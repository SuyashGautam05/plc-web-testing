CTD

The CTD (Count Down) counter decreases its count value each time the input condition changes from FALSE to TRUE. The counter starts counting down from a preset value toward zero.

When the count reaches zero, the counter output becomes TRUE, indicating that the countdown is complete. A reset instruction is required to reload the preset value and restart the counting process.

CTD counters are used in applications such as batch processing, countdown operations, and control systems where a fixed number of operations must be completed.
